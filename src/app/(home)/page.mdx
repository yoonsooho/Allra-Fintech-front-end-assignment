## 🎯 과제의 목적

### 기대하는 역량

- 문제 해결 능력과 창의성
- 코드의 **가독성**, **유지보수성**
- 도구와 프레임워크의 효율적 활용

### 안내 사항

- SEO, 성능 최적화, Next.js 최신 기술 활용 등의 사항은 **필수가 아닙니다**.
- **과제의 완성도보다는 코드 퀄리티와 문제 해결 능력**에 중점을 두어주세요.
- 프레임워크에 익숙하지 않아 구현이 어렵다면, **가능한 범위 내에서 최대한 구현**해주세요.
- 이미 정의된 컴포넌트나 타입은 **자유롭게 변경**하셔도 됩니다.
- **과제에 대한 질문**은 언제든지 환영합니다.

## 📚 과제의 내용

### 1. **올라 마켓** 프로젝트

올라 마켓은 상품을 검색하고 보여주는 가상의 웹 애플리케이션입니다.

- **과제 내용**: 올라 마켓의 이슈를 해결하고, 새로운 기능을 추가해주세요.
- **세부 지침**:
  1. 현재 **실시간 검색기능**이 제대로 동작하지 않고 있습니다. 이를 해결해주세요.
  2. 상품을 클릭했을때 이동되는 **상품 상세 페이지**를 구현해주세요.
     - 화면에 보이는 데이터를 이용하여 **자유롭게 만들어주시면** 됩니다.
     - TailwindCSS 사용은 필수가 아니며, **익숙한 스타일링 방법을 사용해도 좋습니다**.
     - **반응형 디자인을 적용** 해주세요. (모바일, 태블릿, 데스크탑)
